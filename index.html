CloudAtCost 自动重启服务器

该脚本调用CloudAtCost官方API接口执行操作。不存在泄露的问题。更多内容：http://51.ruyo.net/p/1735.html

API调用参数

确定你本地的外网IP

首先确定你本机的IP。
http://pv.sohu.com/cityjson
http://1212.ip138.com/ic.asp
http://whois.pconline.com.cn/
国内好多小的宽带商无固定外网IP,网络情况很复杂!!

登录系统设置

Setting -> 创建API KEY -> API ACCECC IP 填写你本地的外网IP -> 点击UPDATE。

查看Server ID

打开主页，将处于Pedding机器 [i]图标处，鼠标右键 -> [检查] ->在控制台中可见Server ID。 如图 Server ID 是 254795732
